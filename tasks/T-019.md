## Goal

We need to create a skill for ai to understand how the formating of the Ordna application works and structures the data. The skill should also over the config. The skill should be easy to install.

## Acceptance Criteria

- [x] Vendor-neutral `AGENTS.md` template that documents task file format, `.ordna/config.yaml`, and CLI commands.
- [x] `ordna skill install` writes the bundled template into the user's project (defaults to `AGENTS.md` at project root).
- [x] `ordna skill install --from <url>` fetches the skill from a URL — supports the "give the agent a URL" install path.
- [x] `--out <path>` and `--force` (overwrite) options.
- [x] CLI test covers the install path.
- [x] README mentions the skill and the canonical raw URL agents can fetch.
