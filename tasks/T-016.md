## Goal
Can you add a hade/show filter toggle witch to hide show the filter.
Also if you tab when the filter is hidden it should auto show and auto hide
when tabing away again.

## Acceptance Criteria

- [x] TUI: hotkey `f` toggles the left filter sidebar's visibility.
- [x] TUI: when the sidebar is hidden, pressing Tab auto-shows and focuses it (peek mode).
- [x] TUI: pressing Tab again (or Esc) returns focus to the board and auto-hides the sidebar (only if it was opened by Tab — pinned-visible state is preserved).
- [x] TUI: when sidebar is hidden, the board expands to fill the freed width.
- [x] Footer hints reflect the new shortcut (`f hide filter` / `f show filter`).
- [x] Web UI is untouched.

## Notes

Scope is the TUI only (`packages/cli/src/tui`). Web UI (`packages/web`) is out of scope per user clarification.

- Layout: `sidebarWidth` collapses to 0 when hidden so the board area expands.
